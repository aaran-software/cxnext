# codexsun Docs

`apps/docs` is the unified documentation root for the Codexsun platform and its standalone application family.

It documents:

1. reusable framework services
2. shared core business foundations
3. standalone application behavior
4. connector and integration boundaries
5. startup, delivery, and operations
6. platform direction for multi-app delivery from one repository

Primary documents:

1. `apps/docs/STARTUP.md`
2. `ASSIST/Documentation/PROJECT_OVERVIEW.md`
3. `ASSIST/Documentation/ARCHITECTURE.md`
4. `ASSIST/Documentation/SETUP_AND_RUN.md`
5. `ASSIST/Execution/TASK.md`
